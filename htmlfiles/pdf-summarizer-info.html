Generative AI PDF Summarizer

This Generative AI PDF Summarizer leverages the facebook/bart-large-cnn model from Hugging Face to extract and summarize text from PDF files. Users can select a specific page to extract text, edit it if necessary, and generate a concise summary with ease. The intuitive interface simplifies the process with clear options for text extraction and editing.


  • Purpose: Summarizes PDF content using advanced AI models.
  • Model Used: Hugging Face model facebook/bart-large-cnn.
  • Main Functionality:
    1. Extracts text from selected PDF pages.
    2. Summarizes the extracted content.
    3. Allows users to edit extracted text.
  • User Interface
    1. Select page number to extract text.
    2. Button to extract page text.
    3. Editable text area for customization.
